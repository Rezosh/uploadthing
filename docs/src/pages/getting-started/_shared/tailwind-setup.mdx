Wrap your Tailwind config with the `withUt` helper. You can learn more about our
Tailwind helper in the ["Theming" page](/theming#with-tailwindcss)

```tsx copy filename="tailwind.config.ts"
import { withUt } from "uploadthing/tw";

// Your existing Tailwind config
const tailwindConfig = {
  content: ["./src/**/*.{ts,tsx,mdx}"],
  ...
};

export default withUt(tailwindConfig);
```
